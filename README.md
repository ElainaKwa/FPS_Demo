# FPS_Demo

基于 Unreal Engine 5.7 开发

> **`Source/FPS_Demo/` 中的内容是UE第一人称模版自带的代码，请忽略，其中内容不会参与项目的实际编写。** 

FPS_Demo 是一个采用 Gameplay Ability System（GAS）实现武器机制的第一人称射击项目，核心模块为 **`MyFps_Demo`**。
